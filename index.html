Reducing downtime on public bus system using gps and machine learning

A system wherein we use real-time tracking of buses using arduino and gps and gsm sheild and at a later stage machine learning (regression technique) to provide the user back his estimated time of arrival at his preferred bus stop and to the desired destination. A simple express server is used to receive the bus cordinates through Get request (via arduino) and returns data in form of json object which later is used by android app to show busses on the map.
